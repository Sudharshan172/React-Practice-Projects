# Cash Withdrawal Simulator
Simulates ATM-style withdrawals using denomination buttons.

## 🚀 Features
- Balance deduction per click
- Dynamic balance updates with controlled props

## 🧠 Concepts Practiced
- State updates (`useState`)
- Prop usage and reuse through `DenominationItem`

## 🛠️ Technologies
- React + Vite
- Tailwind CSS
